# Rudō (流道) Theme - Unified Color Palette
# Zen Buddhism-inspired design
# Kanso (simplicity), Shizen (naturalness), Seijaku (tranquility)

# ============================================================================
# Core Palette - Primary Colors
# ============================================================================

# Backgrounds
#2d353b  - bg-primary        Main background (warm dark green-gray) - primary UI background
#272e33  - bg-secondary      Alternate background (darker green-gray) - sidebars, panels
#343f44  - bg-dark           Dark black (terminal black, borders, darkest elements)
#3d484d  - bg-mid            Mid-tone dark (widgets, inactive elements, dividers)
#475258  - bg-hover          Medium dark (hover states, borders, interactive elements)

#d3c6aa  - fg-primary        Main foreground (soft warm beige) - primary text color
#859289  - fg-secondary      Secondary foreground (light gray-green) - inactive text
#9da9a0  - fg-bright         Bright white (terminal bright white, highlights, active text)
#868d80  - fg-dim            Bright black (terminal bright black, dimmed text)

#FF8C00  - accent-primary    Saffron orange (primary accent) - Tibetan monk robe color
                            # Use for: active states, highlights, focus, borders, buttons

# ============================================================================
# Color Usage Guidelines
# ============================================================================

# Primary Colors (Most Common)
#   Background:  #2d353b (bg-primary) - Main UI background
#   Background:  #272e33 (bg-secondary) - Sidebars, panels, alternate backgrounds
#   Foreground:  #d3c6aa (fg-primary) - Main text color
#   Accent:      #FF8C00 (accent-primary) - Active states, highlights, focus

# Semantic Colors
#   Error:       #e67e80 (red) - Errors, warnings, destructive actions
#   Success:     #a7c080 (green) - Success states, positive feedback
#   Info:        #7fbbb3 (blue) - Links, informational elements
#   Warning:     #dbbc7f (yellow) - Warnings, highlights

# Interactive States
#   Hover:       #475258 (bg-hover) or #3d484d (bg-mid)
#   Active:      #FF8C00 (accent-primary)
#   Selected:    #3d484d (bg-mid) or #FF8C00 (accent-primary) with transparency
#   Inactive:    #859289 (fg-secondary)

# Borders & Dividers
#   Active:      #FF8C00 (accent-primary)
#   Inactive:    #3d484d (bg-mid) or #4f5b58 (bg-border-inactive)
#   Default:     #343f44 (bg-dark)

# ============================================================================
# Design Philosophy
# ============================================================================

# The theme emphasizes warm, soft colors that are easy on the eyes,
# with the saffron orange (#FF8C00) as the primary accent color inspired
# by Tibetan monk robes, representing the Zen Buddhist aesthetic.

# Color relationships:
# - Backgrounds progress from darkest (#272e33) to lighter (#2d353b) to mid-tones (#3d484d)
# - Foregrounds progress from dim (#868d80) to normal (#d3c6aa) to bright (#9da9a0)
# - Accents use warm colors (saffron, green, teal) for harmony
# - Errors use soft coral red (#e67e80) to be noticeable but not jarring
